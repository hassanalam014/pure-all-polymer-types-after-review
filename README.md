# Pure All Polymer Types After Review

**pure-all-polymer-types-after-review** is a Python-based research repository that collects, reviews, and organizes **thermodynamic and glass-transition parameters for pure polymer systems**.  
The project compiles experimentally validated and literature-reviewed data and provides reusable scripts for analysis, comparison, and visualization of polymer properties.

This repository is primarily intended for **polymer physics, materials science, and thermodynamic modeling research**.

## üß† Supported Polymer Data

Examples of polymers included (depending on reference selection):

- PMMA (Polymethyl methacrylate)
- PS (Polystyrene)
- PC (Polycarbonate)
- PP (Polypropylene)
- PVAc (Polyvinyl acetate)
- PVME (Polyvinyl methyl ether)
- Other reviewed pure polymer systems

Each polymer includes:

- Characteristic pressure (P\*)
- Characteristic temperature (T\*)
- Characteristic density (R\*)
- Atmospheric glass transition temperature (Tg)
- Pressure coefficient of Tg (dTg/dP)
- Experimental Tg vs pressure datasets
